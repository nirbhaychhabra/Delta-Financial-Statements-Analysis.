# Delta-Financial-Statements-Analysis.

This project analyzes Delta Air Lines (DAL) financial statements to assess profitability, liquidity, leverage, and cash flow performance.

Using data from Delta’s 10-K and 10-Q filings, the analysis applies standard financial ratio and trend analysis to evaluate the company’s post-COVID recovery, cost structure, and balance sheet strength.

Key focus areas:
Income statement trends
Balance sheet and leverage
Cash flow sustainability
Core financial ratios

Tools: Excel / Spreadsheet Modeling
Skills: Financial analysis, ratio analysis, business interpretation
